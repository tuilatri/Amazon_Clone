SECTION 2: PURPOSE ALIGNMENT ANALYSIS
================================================================================

WHAT IS AN "OVERVIEW" DASHBOARD?
---------------------------------
An Overview dashboard should provide:
  ✓ Immediate health check (is everything OK?)
  ✓ Key metrics at a glance (numbers, not details)
  ✓ Actionable alerts (anything needs attention?)
  ✓ Quick navigation to deeper sections

An Overview dashboard should NOT provide:
  ✗ Detailed management capabilities
  ✗ Complex analytics or charts
  ✗ Historical trend analysis
  ✗ Configuration or settings
  ✗ Full CRUD operations

APPLICATION TO "FUTURE USAGE" FEATURES:
---------------------------------------
| Feature              | Health Check | At-a-glance | Actionable | Fits? |
|----------------------|--------------|-------------|------------|-------|
| New Customers Today  | ✓            | ✓           | ○          | YES   |
| Active Users (count) | ✓            | ✓           | ○          | YES   |
| User Filters         | ✗            | ✗           | ✓          | NO    |
| Admin Reports        | ✗            | ✗           | ✗          | NO    |

SECTION 3: USER INTENT (ADMIN PERSPECTIVE)
================================================================================

WHAT DOES AN ADMIN CHECK DAILY?
-------------------------------
Morning routine for a typical e-commerce admin:

1. Login → Land on Overview
2. Quick scan: "Any pending orders?" → Order Status Summary
3. Quick scan: "How's revenue?" → Revenue card
4. Quick scan: "Any issues?" → Not currently shown (alerts)
5. Process pending orders → Click status card or go to Order Management
6. Done with Overview

WHEN WOULD AN ADMIN CHECK "FUTURE USAGE" FEATURES?
---------------------------------------------------
| Feature              | Check Frequency | When?                      |
|----------------------|-----------------|----------------------------|
| New Customers Today  | Daily           | Morning scan               |
| Active Users (week)  | Weekly          | Weekly review              |
| User Filters         | As needed       | When investigating a user  |
| Admin Reports        | Weekly/Monthly  | Planning meetings          |

CONCLUSION:
- "New Customers Today" → Daily → Overview ✓
- "Active Users (week)" → Weekly → Overview (simple) or Analytics (detailed)
- "User Filters" → As needed → User Management ✓
- "Admin Reports" → Weekly/Monthly → Sales Analytics ✓
